# NHANES Machine Learning Analysis

This folder is reserved for the second part of the project: an individual-level machine learning analysis based on NHANES data.

## Planned Scope

- prepare and merge NHANES source tables
- define a hypertension target variable
- perform exploratory analysis of individual risk factors
- train and compare classification models
- evaluate model performance and interpret feature importance

## Suggested Structure

- `data/raw` - original NHANES files
- `data/processed` - cleaned modeling datasets
- `notebooks` - EDA and modeling notebooks
- `models` - saved models or related outputs
