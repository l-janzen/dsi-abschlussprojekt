# Dashboard App

This folder is reserved for the third part of the project: the application layer for presenting the project results.

## Planned Scope

- build a Streamlit app for project presentation
- show the project background and data sources
- present the WHO public health findings
- present the NHANES machine learning results
- create a clear final conclusion and limitations section

## Suggested Structure

- `app` - Streamlit application code
- `assets` - static files such as images or exported charts
